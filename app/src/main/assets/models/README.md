# ONNX Model Directory

This directory contains SDXL ONNX models generated by the GitHub Actions workflow.

## Contents (after build)

```
models/
├── text_encoder_1.onnx    # CLIP text encoder 1 (768-dim)
├── text_encoder_2.onnx    # OpenCLIP text encoder 2 (1280-dim)
├── unet.onnx              # SDXL UNet (FP16)
├── vae_decoder.onnx       # VAE decoder
├── tokenizer_1/           # CLIP tokenizer vocabulary
├── tokenizer_2/           # OpenCLIP tokenizer vocabulary
├── scheduler/             # Euler scheduler config
└── manifest.json          # Model metadata
```

## Model Sizes (Approximate)

- text_encoder_1.onnx: ~500 MB
- text_encoder_2.onnx: ~1.4 GB
- unet.onnx (FP16): ~5.1 GB
- vae_decoder.onnx: ~200 MB

Total: ~7.2 GB

## Note

These files are NOT committed to git. They are generated during the CI build process
and embedded into the APK as assets.
